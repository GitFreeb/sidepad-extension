                              SIDEPAD

Боковая панель Chrome для ведения списков задач в формате Markdown.
Работает как нативный Side Panel — не перекрывает страницу,
всегда под рукой.

ЧТО ЭТО ТАКОЕ
--------------
Sidepad — расширение для Chrome, которое открывает боковую панель
со списком задач. Панель работает поверх любой вкладки браузера,
не закрывая её. Задачи хранятся локально в браузере, читаются и
сохраняются в формате Markdown (.md).

Расширение не требует регистрации, не отправляет данные в сеть,
не имеет зависимостей и не нуждается в сборке.


ВОЗМОЖНОСТИ
-----------

  Задачи
  - Добавление задач с текстом, URL-ссылкой и привязкой к секции
  - Отметка задач как выполненных (чекбокс)
  - Редактирование текста задачи по клику
  - Удаление пользовательских задач
  - Копирование текста задачи в буфер обмена одной кнопкой
  - Перемещение задачи в другую секцию кнопкой → (появляется при наведении)
  - Выполненные задачи автоматически перемещаются в конец секции

  Секции
  - Группировка задач по именованным секциям
  - Добавление новых секций через форму
  - Сворачивание/разворачивание секций кликом по заголовку
  - Кнопка «▸ свернуть все / ▾ развернуть» для управления всеми секциями
  - Переименование секции кнопкой ✎ (появляется при наведении на заголовок)
  - Удаление секции вместе со всеми её задачами
  - Автосворачивание секции, когда все задачи в ней выполнены
  - Счётчик выполненных задач в заголовке каждой секции

  Вкладки
  - Одновременная работа с несколькими списками в разных вкладках
  - Каждый импортированный файл открывается в отдельной вкладке
  - Переименование вкладки двойным кликом по её имени
  - Индикатор скрытых вкладок (+N) при горизонтальном переполнении
  - Закрытие вкладки автоматически сохраняет список в Downloads
  - При закрытии последней вкладки список сбрасывается к пустому состоянию

  Импорт и экспорт
  - Загрузка .md и .txt файлов через системный диалог
  - Сохранение текущего списка в .md через диалог «Сохранить как»
  - Кнопка ↓auto для включения/выключения автосохранения в Downloads
  - Поддержка чекбоксов [x] и [ ] в Markdown
  - URL в задачах извлекаются и становятся кликабельными ссылками

  Внешний вид
  - Тёмная и светлая тема (переключатель в шапке)
  - Два языка интерфейса: русский и английский (переключатель RU/EN)
  - Drag & Drop для задач и секций


УСТАНОВКА
---------
1. Скачать или клонировать репозиторий
2. Открыть в Chrome: chrome://extensions
3. Включить Режим разработчика (переключатель в правом верхнем углу)
4. Нажать «Загрузить распакованное расширение»
5. Выбрать папку Sidepad/
6. Нажать на иконку расширения в панели Chrome — откроется боковая
   панель

  Примечание: после изменений в файлах расширения нажмите кнопку
  перезагрузки на карточке расширения в chrome://extensions, затем
  закройте и откройте боковую панель заново.


БЫСТРЫЙ СТАРТ
-------------
В папке расширения находится файл example.md — готовый список с
задачами, который демонстрирует все возможности Sidepad: секции,
ссылки, перетаскивание, импорт и экспорт.

Загрузить пример можно двумя способами:

  - Кнопка в пустом списке: при первом запуске в пустом default.md
    отображается кнопка «Загрузить пример →» — нажми её, и example.md
    откроется сразу.
  - Вручную: нажми ↑ загрузить и выбери файл example.md из папки
    Sidepad/.

После загрузки default.md автоматически закрывается.


ИНТЕРФЕЙС
---------

  ┌──────────────────────────────────────┐
  │ ✳ default.md × ✳ work.md ×  ↓auto RU ◐ │  ← Вкладки + кнопки
  ├──────────────────────────────────────┤
  │ ~/tasks/ default.md    —  3 / 10         │  ← Breadcrumb + счётчик
  ├──────────────────────────────────────┤
  │ + новая секция...                        │  ← Поле новой секции
  │ + новая задача...                        │  ← Поле новой задачи
  │   url (необязательно)                    │  ← Поле URL
  │ [Выбор секции ▾]  [+ добавить]           │  ← Список + кнопка
  ├──────────────────────────────────────┤
  │ ↑ загрузить ↓ выгрузить ▸ свернуть ✕ очистить│  ← Тулбар
  ├──────────────────────────────────────┤
  │ ▸ Раздел 1               2/5   ›         │  ← Заголовок (свёрнут)
  │ ▾ Раздел 2               0/3   ›         │  ← Заголовок (развёрнут)
  │   ⠿ ☐  Задача со ссылкой  ⧉ → ×         │  ← Задача (кнопки при наведении)
  │         https://example.com              │
  │   ⠿ ☑  Выполненная задача   ⧉ ×         │  ← Выполненная задача
  └──────────────────────────────────────┘

  Элементы интерфейса:

  ✳ название.md ×    Вкладка. × — закрыть с сохранением в Downloads
  RU / EN            Переключатель языка интерфейса
  ◐ / ●              Переключатель тёмной/светлой темы
  ~/tasks/ имя.md    Breadcrumb — имя текущей вкладки
  N / M              Счётчик: выполнено / всего задач
  ↑ загрузить        Открыть .md файл в новой вкладке
  ↓ выгрузить        Сохранить текущий список в файл
  ✕ очистить         Очистить список (с автосохранением в Downloads)
  ⠿                  Ручка для перетаскивания задачи
  ⧉                  Копировать текст задачи в буфер
  × на задаче        Удалить задачу
  × на секции        Удалить секцию со всеми задачами (при наведении)
  ✎ на секции        Переименовать секцию (появляется при наведении)
  → на задаче        Переместить задачу в другую секцию (при наведении)
  N/M в заголовке    Выполнено / всего в этой секции
  ↓auto              Включить/выключить автосохранение в Downloads
  ▸ свернуть /
  ▾ развернуть       Свернуть или развернуть все секции сразу


РАБОТА С ЗАДАЧАМИ
-----------------

  Добавить задачу
  1. Заполнить поле «+ новая задача...»
  2. При необходимости указать URL в поле «url (необязательно)»
  3. Выбрать секцию в выпадающем списке
  4. Нажать «+ добавить» или клавишу Enter

  Новая задача добавляется в начало выбранной секции.

  Отметить как выполненную
  Нажать на чекбокс слева от задачи. Выполненная задача перемещается
  в конец своей секции и визуально выделяется. Если все задачи в
  секции выполнены — секция автоматически сворачивается.
  Повторное нажатие снимает отметку.

  Редактировать задачу
  Кликнуть по тексту задачи. Текст станет редактируемым полем.
  Для сохранения — Enter или клик вне поля. Для отмены — Escape.

  Удалить задачу
  Нажать кнопку × справа от задачи. Кнопка доступна только для
  задач, добавленных вручную.

  Скопировать текст
  Нажать кнопку ⧉ — текст задачи скопируется в буфер обмена,
  появится всплывающее уведомление «⧉ copied».

  Переместить в другую секцию
  Навести курсор на задачу — появится кнопка → справа. Нажать её
  и выбрать нужную секцию из меню. Задача переместится в начало
  выбранной секции.

  Перейти по ссылке
  Если у задачи есть URL, он отображается строкой ниже текста.
  Клик по ссылке откроет её в новой вкладке браузера.


РАБОТА С СЕКЦИЯМИ
-----------------

  Добавить секцию
  1. Заполнить поле «+ новая секция...»
  2. Нажать «+ добавить» или клавишу Enter в поле секции

  Новая секция добавляется в начало списка и становится доступной
  в выпадающем списке при добавлении задач.

  Примечание: если заполнены оба поля (и секция, и задача) —
  при нажатии «+ добавить» добавятся оба элемента.

  Свернуть / развернуть секцию
  Кликнуть по заголовку секции. Стрелка › указывает направление.
  Секция автоматически сворачивается, когда все её задачи выполнены.
  Кнопка «▸ свернуть все» в тулбаре сворачивает все секции одним
  нажатием. Повторное нажатие — «▾ развернуть» — раскрывает их.

  Переименовать секцию
  Навести курсор на заголовок секции — появится кнопка ✎ справа.
  Нажать её — заголовок превратится в текстовое поле. Для сохранения
  — Enter или клик вне поля. Для отмены — Escape.
  Если новое имя совпадает с уже существующей секцией — задачи из
  обеих секций объединятся.

  Удалить секцию
  Навести курсор на заголовок секции — появится кнопка × справа.
  Нажать её. Секция и все её задачи удаляются без подтверждения.


ВКЛАДКИ
-------
Каждый импортированный файл открывается как отдельная вкладка.
Вкладки позволяют одновременно работать с несколькими списками.

  Переключиться между вкладками
  Кликнуть по нужной вкладке в панели сверху. Состояние каждой
  вкладки сохраняется при переключении.

  Закрыть вкладку
  Нажать × на вкладке. Текущая версия списка автоматически
  сохранится в папке Downloads в формате название_ГГГГ-ММ-ДД_ЧЧ-ММ.md.
  При закрытии последней вкладки список не исчезает — содержимое
  сбрасывается к пустому состоянию default.md.

  Переименовать вкладку
  Дважды кликнуть по имени вкладки — оно станет редактируемым.
  Расширение .md добавляется автоматически, редактируется только имя.
  Для подтверждения — Enter или клик вне поля. Для отмены — Escape.


ИМПОРТ И ЭКСПОРТ
----------------

  Импорт файла
  Нажать ↑ загрузить. Открывается системный диалог выбора файла
  (форматы .md, .txt). После выбора файл разбирается и открывается
  в новой вкладке.

  Формат определяется автоматически.

  Markdown (есть строки "- " или "* "):

      # Название списка

      ## Название секции

      - [ ] Невыполненная задача
      - [x] Выполненная задача
      - [ ] Задача со ссылкой  https://example.com

      ## Другая секция

      - [ ] Ещё задача

  - Строки "## Заголовок" становятся секциями
  - Строки "- [ ]", "- [x]", "* " становятся задачами
  - URL в строке задачи автоматически становится кликабельной ссылкой
  - Строки #, |, `, ! и строки длиннее 120 символов игнорируются

  Простой текст (нет "- " / "* " буллетов — нумерованные списки, заметки):

      НАЗВАНИЕ СЕКЦИИ

      1. Первая задача
      2. Вторая задача

      Другая секция:
      * Третья задача
      • Четвёртая задача

  - СТРОКА ЗАГЛАВНЫМИ БУКВАМИ (≤ 60 символов) → секция
  - Строка, заканчивающаяся на : (≤ 60 символов) → секция
  - Заголовок, подчёркнутый --- или === → секция
  - Пункты 1. / 1) → задачи
  - Маркеры * / • → задачи
  - Любая обычная строка → задача
  - Метки [x], ✓, ✔ → задача помечается как выполненная

  Экспорт в .md файл
  Нажать ↓ выгрузить. Откроется диалог «Сохранить как» с текущим
  именем файла. Экспортированный файл содержит все секции и задачи
  с отметками [x] / [ ].

  Автосохранение
  Управляется кнопкой ↓auto в панели вкладок. По умолчанию включено.
  При выключении кнопка визуально затухает — автосохранение не
  происходит.

  При включённом автосохранении файл сохраняется:
  - При закрытии вкладки через ×
  - При нажатии ✕ очистить (перед удалением содержимого)

  Вкладка example никогда не сохраняется автоматически. Вкладка
  default сохраняется без диалога подтверждения.

  Файл сохраняется в Downloads с именем название_ГГГГ-ММ-ДД_ЧЧ-ММ.md.
  Если файл уже содержит дату в имени — она заменяется новой, а не
  добавляется повторно.


ПЕРЕТАСКИВАНИЕ
--------------

  Перетаскивание задач
  Захватить задачу за ручку ⠿ и перетащить:
  - На другую задачу — разместится до или после неё (в зависимости
    от позиции курсора)
  - На заголовок секции — задача переместится в эту секцию (в конец)

  Примечание: альтернатива drag & drop — кнопка → при наведении на
  задачу: задача перемещается в начало выбранной секции.

  Перетаскивание секций
  Захватить заголовок секции и перетащить на другой заголовок —
  секция со всеми задачами переместится до или после целевой секции.

ХРАНЕНИЕ ДАННЫХ
---------------
Все данные хранятся в chrome.storage.local — локально в браузере,
без передачи в сеть.

  Структура хранилища:

      {
        "tabs": [
          {
            "id": "tab_1234567890",
            "title": "work",
            "tasks": [ ... ],
            "sections": ["Приоритет", "Бэклог"]
          }
        ],
        "activeTabId": "tab_1234567890",
        "lightMode": false,
        "lang": "ru",
        "autoSave": true
      }

  Данные сохраняются при каждом изменении и восстанавливаются при
  следующем открытии панели. Старый формат данных (до введения
  вкладок) автоматически мигрирует в новый при первом открытии.


ТЕХНИЧЕСКИЕ ДЕТАЛИ
------------------

  Manifest версия    V3
  Разрешения         storage, sidePanel
  Зависимости        нет
  Сборка             не требуется
  Стек               Vanilla JS, HTML, CSS
  Точка входа        popup.html → popup.js
  Service Worker     background.js

  Структура папки расширения:

      Sidepad/
      ├── manifest.json       — конфигурация расширения
      ├── background.js       — service worker, открытие панели по клику
      ├── popup.html          — разметка интерфейса
      ├── popup.css           — стили
      ├── popup.js            — вся логика
      ├── default.md          — пустой список по умолчанию
      ├── example.md          — пример использования для новых пользователей
      ├── README.md           — документация (русский)
      ├── README.en.md        — документация (English)
      ├── INFO.ru.txt         — документация plain text (русский)
      ├── INFO.en.txt         — документация plain text (English)
      └── icons/
          ├── icon16.png
          ├── icon32.png
          ├── icon48.png
          └── icon128.png

  Конвенция ID задач: встроенные задачи имеют id ≤ 10000,
  пользовательские — id > 10000 (вычисляется как Date.now() + 10001).
  Кнопка удаления отображается только для задач с id > 10000.
