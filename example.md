## Секции

- [ ] Кликни на заголовок секции — она свернётся
- [ ] Нажми ▸ свернуть в тулбаре — свернёт все секции сразу
- [ ] Перетащи заголовок секции, чтобы поменять их порядок
- [ ] Добавь новую секцию через поле «+ новая секция...» сверху
- [ ] Наведи на заголовок — появятся ✎ (переименовать) и × (удалить секцию)

## Задачи со ссылками

## Вкладки и файлы

- [ ] Нажми ↑ загрузить — откроешь любой .md файл в новой вкладке
- [ ] Нажми ↓ выгрузить — сохранишь список как .md файл
- [ ] Двойной клик по названию вкладки — переименуй её
- [ ] Кнопка ↓auto вверху — включает и выключает автосохранение в Downloads
- [ ] Закрой вкладку через × — список сохранится в Downloads (если ↓auto включён)
